# Resume Analyzer using IBM Watsonx Prompt Lab

## Overview

This project demonstrates few-shot prompting using IBM Watsonx Prompt Lab.

The model analyzes a student's education, skills, and projects and recommends a suitable career path.

## Features

* Career Path Recommendation
* Strength Analysis
* Weakness Analysis
* Skill Gap Identification
* Learning Recommendations
* Overall Profile Rating

## Prompting Technique

Few-Shot Prompting

Multiple input-output examples are provided to teach the model how to evaluate different student profiles.

## Sample Output

* Best Career Path
* Why this Career Path is suitable
* Strengths
* Weaknesses
* Skills to Learn Next
* Overall Rating

* Few-Shot Prompting
