# Prompt Engineering Lab

A collection of Prompt Engineering projects built using IBM Watsonx Prompt Lab.

## Projects

### 1. Text Summarizer
Converts long stories into concise bullet-point summaries.

### 2. Resume Analyzer
Analyzes student profiles and recommends suitable career paths with personalized feedback.

## Technologies

- IBM Watsonx.ai
- IBM Cloud
- Prompt Engineering
- Foundation Models

## Prompting Techniques Used

- Zero-Shot Prompting
- One-Shot Prompting
- Few-Shot Prompting
