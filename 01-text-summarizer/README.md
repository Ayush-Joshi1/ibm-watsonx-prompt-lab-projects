# Text Summarizer using IBM Watsonx Prompt Lab

## Overview

This project demonstrates text summarization using IBM Watsonx Prompt Lab.

A long story is provided as input and the model generates a concise summary highlighting the most important events.

## Prompting Technique

One-Shot Prompting

The model is provided with an example story and its summary before generating summaries for new stories.

## Input

A long-form story or article.

## Output

A concise bullet-point summary containing the key events.
